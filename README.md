# DoLog 🚀

DoLogis a modern productivity app built to help users stay organized, consistent, and accountable. Instead of acting as just another to-do list, it focuses on tracking progress and encouraging users to complete their tasks through visual insights and productivity analytics.

The app includes a **Dashboard** for viewing upcoming tasks and daily progress, a **Todo List** for managing tasks, a **Calendar** for planning schedules, and a **Profile** to monitor personal productivity.

One of its key features is a **GitHub-style monthly activity heatmap**, allowing users to visualize their consistency by showing how many tasks were completed each day. This makes it easy to identify productive streaks and areas that need improvement.

## Features

* Create, edit, and delete tasks
* Organize tasks with an intuitive interface
* Dashboard with upcoming tasks and progress overview
* Calendar for better planning
* GitHub-style monthly activity heatmap
* Personal productivity profile
* Clean and responsive UI
